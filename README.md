# 📖 Overview

StockVision AI is a modern AI-driven investing platform designed specifically for Indian investors.

The platform combines:

✅ AI Portfolio Analysis

✅ Market Forecasting

✅ Real-Time Market Tracking

✅ AI-Powered Insights

✅ Stock Screening

✅ IPO Intelligence

✅ Paper Trading

✅ Portfolio Management

✅ Watchlists

✅ Smart Alerts

into one unified platform.

Instead of using multiple websites for charts, portfolio tracking, research, IPO analysis, forecasts, and market news, StockVision AI brings everything together in a single intelligent dashboard.

                 👤 User
                    │
                    ▼
      ⚛️ React + TanStack Start
                    │
                    ▼
          📡 API Layer / Services
                    │
      ┌─────────────┼─────────────┐
      ▼                           ▼
🤖 Google Gemini AI       📊 Market APIs
      │                           │
      └─────────────┬─────────────┘
                    ▼
           🗄 Supabase Backend
                    │
                    ▼
          🔐 Authentication
                    │
                    ▼
         ☁️ Cloudflare Workers

# 🚀 Installation

```bash
git clone https://github.com/AvadhKalathiya/stockvision-ai.git

cd stockvision-ai

npm install

npm run dev
```

Production Build

```bash
npm run build
```

Preview

```bash
npm run preview
```

# 🔐 Environment Variables

Create a .env file:

```env
SUPABASE_URL=your_supabase_url

SUPABASE_PUBLISHABLE_KEY=your_key

VITE_SUPABASE_URL=your_supabase_url

VITE_SUPABASE_ANON_KEY=your_key

GEMINI_API_KEY=your_key

NEWS_API_KEY=your_key

GNEWS_API_KEY=your_key

FINNHUB_API_KEY=your_key
```
